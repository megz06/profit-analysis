# Profit Analysis

## About This Project
Analysis of profit across categories,
region and sub-categories using 9,994 sales records.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
1. Technology has highest profit
2. Furniture has lowest profit
3. West region is most profitable
4. Central region need improvement
5. Copiers is most profitable sub-categoy

## Skills Demonstrated
- Groupby analysis
- Data visualization
- Business insight generation
